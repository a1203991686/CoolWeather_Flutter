# cool_weather

## 简介
一个Flutter练手的天气预报Demo。

## 使用的东西
主要使用了一些基本的Flutter Widget，并使用了路由管理、包管理、HttpClient、Dio、Json转Dart实体类、SharedPreferences、异步。
- 基础组件
    - Text
    - Image
- 布局类组件
    - Row
    - Column
    - Expanded
    - Stack
    - Align
- 容器类组件
    - Padding
    - ConstrainedBox
    - Container
    - Scaffold
- 可滚动组件
    - SingleChildScrollView
    - ListView
    - GridView
- 功能性组件
    - FutureBuilder
- 包
    - material.dart
    - dart:convert
    - dart:io
    - shared_preferences
    - json_annotation
    - json_serializable
